# Hackathon Submission Checklist

- Public GitHub repository with clean README.
- Contracts deployed on Avalanche Fuji with addresses and explorer links.
- Live frontend URL.
- Demo video under expected hackathon length.
- Wavy Node integration proof with API-key server path and HMAC-protected integration routes.
- Screenshots of allow/block transfer results.
- Architecture diagram and product explanation.
- Clear limitation note: demo/sandbox, no real public offering, no custody, no brokerage.
- Business model and sponsor alignment section.

## One-Minute Pitch

Retail investors in LatAm are starting to see tokenized financial assets, but regulated distribution is still fragmented. AvaReg is an On-Chain Agent that lets users discover products, complete profiling, pass Wavy wallet risk scoring and route requests to responsible institutions. Avalanche records referral evidence and transfer gate decisions, while execution and custody stay with licensed entities.
